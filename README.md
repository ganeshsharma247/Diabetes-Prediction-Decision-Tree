# 🩺 Diabetes Prediction using Decision Tree Classifier

A Machine Learning project that predicts whether a patient is likely to have diabetes based on diagnostic medical features. This project includes data preprocessing, exploratory data analysis (EDA), hyperparameter tuning, model evaluation, and model visualization using a Decision Tree Classifier.

---

## 📌 Project Overview

The objective of this project is to build a machine learning classification model that can predict diabetes using patient health information. The workflow follows a complete machine learning pipeline, including data cleaning, exploratory data analysis, model training, hyperparameter tuning, and evaluation.

---

## 🎯 Problem Statement

Early detection of diabetes is important for timely medical intervention. This project aims to classify patients into diabetic or non-diabetic categories using medical attributes such as glucose level, BMI, blood pressure, insulin level, age, and other clinical measurements.

---

## 📂 Dataset

- **Dataset:** Pima Indians Diabetes Dataset
- **Source:** Kaggle
- **Records:** 768
- **Features:** 8 Input Features + 1 Target Variable

### Features

- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age

**Target Variable**

- 0 → No Diabetes
- 1 → Diabetes
